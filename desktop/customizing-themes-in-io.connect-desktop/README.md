# **Customize Themes in io.Connect Desktop**

_Author: [Georgi Georgiev](https://community.interop.io/u/georgi_g)_

This guide explains how theming works in io.Connect Desktop and how to build your own theme pack from scratch.

It covers:
- The structure of a theme pack (manifest + color tokens + assets).
- How io.Connect loads themes, fallback rules, and how overrides are resolved.
- The full list of configurable UI areas (window chrome, panels, buttons, lists, notifications, etc.).
- How to preview and test changes during development.
- Packaging and distributing a theme so it can be selected by users.
